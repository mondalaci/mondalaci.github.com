Unfancify Your Code in WordPress

2007-07-25 00:37:33 GMT

Wordpress has a nasty feature called smart quotes or curly quotes. It transforms " " quote pairs into &ldquo; (“) &rdquo; (“) HTML entities. It may look good with ordinary text, but it seriously messes up code.

The Unfancy Quote WordPress plugin was a breath of fresh air to me regarding this pain. I can only recommend it to every developers out there!
